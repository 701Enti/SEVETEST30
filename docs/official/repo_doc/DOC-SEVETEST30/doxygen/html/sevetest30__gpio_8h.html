◆ ext_io_level_service()

esp_err_t ext_io_level_service ( )

扩展GPIO电平值服务，读写通用，当有电平变化，系统中断触发会自动调用该函数，所以主要用来写操作

写：先对ext_io_value_data这个全局公共结构体变量修改值，等待系统大循环调用该函数写入IO电平

读：（当中断读取使能，只要电平变化系统中断触发，将全自动读取，一般无需调用）如果需要主动读取：先调用该函数刷新ext_io_value_data，之后直接读取ext_io_value_data内的成员变量值 *‍/

返回
[ESP_OK 成功]
[ESP_ERR_INVALID_ARG 参数错误]
[ESP_FAIL 发送命令时发现问题, TCA6416A未应答]
[ESP_ERR_INVALID_STATE I2C driver 未安装或没有运行在主机模式]
[ESP_ERR_TIMEOUT 操作超时因为总线忙]

◆ ext_io_mode_service()

esp_err_t ext_io_mode_service ( )

扩展GPIO输入输出模式服务

写：先对ext_io_mode_data这个全局公共结构体变量写入需要修改的值，再调用该函数控制IO输入输出模式

读：直接读取ext_io_mode_data内的成员变量值，不需要调用该函数 *‍/

返回
[ESP_OK 成功]
[ESP_ERR_INVALID_ARG 参数错误]
[ESP_FAIL 发送命令时发现问题, TCA6416A未应答]
[ESP_ERR_INVALID_STATE I2C driver 未安装或没有运行在主机模式]
[ESP_ERR_TIMEOUT 操作超时因为总线忙]

)

全局GPIO初始化,包括内部IO和扩展IO

参数
p_mode缓存位置-模式配置(比如定义一个TCA6416A_mode_t结构体作为缓存位置导入,之后可直接改变结构体成员值来控制扩展IO)
p_level缓存位置-电平数据(比如定义一个TCA6416A_level_t结构体作为缓存位置导入,之后可直接改变结构体成员值来控制扩展IO)
返回
[ESP_OK 成功]
[ESP_ERR_INVALID_ARG 输入参数错误 / 选择了错误的GPIO]

[ESP_ERR_TIMEOUT 操作超时因为总线忙]
[ESP_ERR_NO_MEM 安装ISR服务时发现内存不足]
[ESP_ERR_NOT_FOUND 没有找到特定FLAG的空闲interrupt]
