)

磁感应强度数据转换为角数据(输出数据的单位由输入参数unit设置)

src导入磁感应强度数据
dest转换结果保存到dest

[ESP_OK 转换完成并保存到dest]
[ESP_ERR_INVALID_ARG 参数错误, src为NULL / dest为NULL / 选择了不支持的单位(输入的unit值异常)]

GS_magnetic_flux_density_data_t *  dest 
)

将output(未处理的源数据)转换为磁感应强度数据(单位: uT)

dest转换结果保存到dest
