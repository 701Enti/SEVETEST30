)

磁感应强度数据转换为角数据(输出数据的单位由输入参数unit设置)

参数
unit选择输出数据的单位,这是一个枚举类型
src导入磁感应强度数据
dest转换结果保存到dest
注解
为了获取角数据,需要磁感应强度数据,这意味着需要先调用to_magnetic_flux_density_data()获取
返回
[ESP_OK 转换完成并保存到dest]
[ESP_ERR_INVALID_ARG 参数错误, src为NULL / dest为NULL / 选择了不支持的单位(输入的unit值异常)]
