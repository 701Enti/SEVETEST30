三色分离方式 取模适配PCtoLCD2002

取模说明：从第一行开始向右每取8个点作为一个字节，如果最后不足8个点就补满8位。

取模顺序是从高到低，即第一个点作为最高位。如*----—取为10000000

RGB三色分离方式绘制,只支持单色绘制,之后ledarray_set_and_write需要被调用才可显示

取模方式请参考头文件

参数
x图案横坐标(无范围限制，超出不显示)，灯板左上角设为原点（1，1），由左到右绘制
y图案纵坐标(无范围限制，超出不显示)，灯板左上角设为原点（1，1），由上到下绘制
breadth图案宽度（已定义的：FIGURE-数字 RECTANGLE_MATRIX(x)-生成的矩形 ）
p导入字模指针
byte_number总数据长度(Byte)
in_color注入颜色 （RGB）
change亮度调整（1-100）警告:过高的调整幅度可能导致色彩失真
